Currently, PjECRF is far from being finished however the Chinese language in this project is being developed to the point to make you comfortable with the learning process.

Overall, other languages will be implemented sooner or later, but for now this project is still in progress and the goal is to make the language learning more enjoyable and less frustating.

It is scientifically proven that by using the reptition method one can make strong memory connections thus memorizing things more efficiently. This application use reptition method effectively to help you learn more quickly.

Although we'll try our best to assist you in anyway possible, however it's also necessary to practice daily and to be fair it's still not enough to use this application instead we will recommend you to use the full application for much better results.

For now only the demo is available to run which is as simple as to download and run, but if you're coming from some background in programming you can also play with the code to make your contributions.
